# 在 FunC 上開發 TON 智能合約的課程

本課程介紹如何在 FunC 上開發 TON 的智能合約，課程以 .md 格式附上代碼。我們的社群在[這裡](https://t.me/ton_learn)。課程首先是用[俄語]((https://github.com/romanovichim/TonFunClessons_ru/))完成的，在此，然後我會逐步將其翻譯成中文，如果您想幫助，請參閱以下的捐贈地址。
